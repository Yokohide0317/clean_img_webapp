# clean_img_webapp
https://github.com/bilibili/ailab をWebから簡単に動かせるようにFlask実装

## Get Start
- envでpython3.8以上の環境作成を推奨

```
bash setup.sh
python -m pip install -r requirements.txt
python app.py
```

## ダウンロードが"Network Error"になる
- resumeしたら上手くいく
- localで動かしているなら、`static/imgs/clean` 下に保存されてます。
